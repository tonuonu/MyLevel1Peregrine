## L1 Requirements

Per Tripoli certification standards:

- [x] Tripoli membership (Senior, insured)
- [x] H or I class motor selected
- [x] Rocket built by flyer
- [ ] Successful flight with recovery
- [ ] Witnessed by TAP/Prefect
