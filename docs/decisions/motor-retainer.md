# Motor Retainer

## Context

Kit does not include motor retention. Options:

- AeroPack 38L Engine Retainer (commercial)
- Fins & Fire 38mm Retainer (commercial)
- Masking tape (traditional method)
- 3D printed retainer

## Decision

3D printed retainer from [Thingiverse model #6780161](https://www.thingiverse.com/thing:6780161)

## Implementation

- **Material**: ASA (weather/UV resistant, good heat tolerance)
- **Attachment**: Epoxied to aft end of motor mount tube

## Reasoning

- Cost effective
- Available immediately (no shipping wait)
- ASA provides adequate heat resistance for motor exhaust
- Can reprint if damaged
