# Style and Conventions

## Documentation Standards
- All measurements in metric (imperial retained only when from original source)
- Formulas shown with variable definitions
- Photos named descriptively: `YYYYMMDD-description.jpg`
- Flight data recorded with altimeter downloads

## Markdown Features Available
- MathJax LaTeX: `$$formula$$` or `$inline$`
- Mermaid diagrams: fenced with `mermaid`
- Admonitions: `!!! note "Title"`
- Tabbed content: `=== "Tab 1"`
- Code blocks with copy button

## File Organization
- specifications/ - Rocket physical specs
- construction/ - Build documentation
- calculations/ - Engineering formulas
- simulations/ - OpenRocket data
- flight/ - Checklists and logs
- photos/ - Build images
